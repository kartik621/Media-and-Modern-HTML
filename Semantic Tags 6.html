5 Semantic Tags

  1. < header >: The < header > tag represents the introductory content or a container for the introductory content of a section or page. It typically contains the site logo, site title, navigation menus, or other related elements.

  2. < nav >: The < nav > tag defines a section of navigation links, such as a menu, within the document. It is used to group navigation elements that direct users to different parts of the website or other related pages.

  3. < main >: The < main > tag identifies the main content area of a document. It should contain the primary content that is unique to the document, excluding any content that is repeated across a set of documents, such as site navigation or headers.

  4. < article >: The < article > tag represents a self-contained composition in a document, such as a blog post, news article, forum post, or a comment. It should make sense when read independently and can be syndicated or distributed independently from the rest of the page.

  5. < footer >: The < footer > tag defines the footer section of a document or a section. It typically contains information about the author, copyright information, links to related documents, or other relevant data. It appears at the bottom of the document or a section.
